Para ejecutar el programa deberá tener instalado python y su respectivo compilador. Le recomiendo que para ello vaya a las páginas oficiales de este lenguaje y los descargue desde allí.

Una vez descargados podrá ejecutar el programa desde el terminal, accediendo a la carpeta y ejecutando el archivo `Main.py`
